Calibration

Calibration converts pixel measurements to real-world units (micrometers). Accurate calibration is essential for meaningful tubercle diameter and spacing measurements.

The Default Estimate

Important: When no calibration has been set, the system displays a default estimate of 0.14 µm/pixel. This value is specific to a particular imaging setup and may not be accurate for your images.

Where does 0.14 µm/pixel come from?

This estimate is based on:

  • Magnification: 700x
  • Image width: 1024 pixels
  • Field of view: ~140 µm across

The calculation: 140 µm ÷ 1024 pixels = 0.137 µm/pixel (rounded to 0.14)

When is 0.14 µm/pixel accurate?

This estimate is only reasonable if your images were captured with similar settings. If your images have different dimensions or magnification, the actual calibration will differ significantly:

Image Width At 700x Magnification
1024 px ~0.14 µm/pixel
2048 px ~0.07 µm/pixel
4096 px ~0.035 µm/pixel

Different magnifications also change the calibration proportionally. Always verify calibration using a scale bar when available.

Calibration Methods

1. Graphical Scale Bar Measurement (Recommended)

The most accurate method when your image contains a scale bar:

  1. Click "Manually Draw Scale Bar"
  2. Click the start of a known scale bar in the image
  3. Click the end of the scale bar
  4. Enter the known length (e.g., 10 µm)

This method directly measures pixels against a known reference in your specific image.

2. Scale Bar Values

If you know the scale bar dimensions:

  1. Enter the scale bar length in µm
  2. Enter the scale bar length in pixels
  3. Click Apply

3. Direct Input

If you know the exact µm/pixel value from your microscope settings:

  1. Enter the value directly (e.g., 0.165)
  2. Click Apply

4. Auto-Estimate

Uses the default 0.14 µm/pixel estimate. Only use this if:

  • Your image has no scale bar
  • You don't know the microscope settings
  • You only need approximate measurements

Calibration Status Indicators

Indicator Meaning
Estimated Using default 0.14 µm/pixel - measurements may be inaccurate
Measured Calibration set via scale bar measurement
Manual Calibration entered directly or via scale bar values
Loaded Calibration loaded from saved annotations

Persistence

Calibration is saved with your annotations. When you load an image that has saved annotations, the calibration will be restored automatically.

Verification

To verify your calibration is correct:

  1. Run extraction on a few tubercles
  2. Check the Data tab for diameter values
  3. Compare against expected values from literature or known samples

Typical ganoid fish scale tubercle diameters range from 3-15 µm depending on species.
