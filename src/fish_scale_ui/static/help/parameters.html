Parameter Reference

This guide explains the parameters used for tubercle detection and measurement.

Detection Parameters

Detection Method

The algorithm used to detect tubercles in the image.

Method Description Best For
log Laplacian of Gaussian - detects bright circular spots Most images (default)
dog Difference of Gaussian - faster approximation of LoG Large images where speed matters
ellipse Threshold segmentation with ellipse fitting High-contrast images
lattice Hexagonal lattice-aware detection Regularly arranged tubercles

Threshold

Range: 0.01 - 0.50 (default: 0.05)

Detection sensitivity for blob detection. Lower values detect more blobs (more sensitive), higher values detect fewer (more selective).

  • Lower (0.01-0.03): More sensitive, may include noise
  • Medium (0.04-0.08): Balanced for most images
  • Higher (0.10-0.20): Only strong features, may miss faint tubercles

Min Diameter

Range: 0.5 - 20.0 micrometers (default: 2.0)

Minimum expected tubercle diameter. Blobs smaller than this are filtered out.

Typical ranges by genus:

  • Polypterus: 1.5 - 3.0 micrometers
  • Lepisosteus: 3.0 - 6.0 micrometers
  • Paralepidosteus: 5.0 - 7.0 micrometers
  • Atractosteus: 5.0 - 10.0 micrometers

Max Diameter

Range: 1.0 - 50.0 micrometers (default: 10.0)

Maximum expected tubercle diameter. Blobs larger than this are filtered out.

Circularity

Range: 0.0 - 1.0 (default: 0.5)

Minimum circularity filter. 1.0 = perfect circle, 0.0 = any shape.

  • 0.0-0.3: Accept elongated/irregular shapes
  • 0.4-0.6: Roughly circular shapes (recommended)
  • 0.7-1.0: Only near-perfect circles

Note: Set to 0 for scanned PDF images where boundary definition is poor.

Preprocessing Parameters

CLAHE Clip Limit

Range: 0.01 - 0.20 (default: 0.03)

Contrast Limited Adaptive Histogram Equalization enhances local contrast.

The clip limit controls contrast amplification:

  • Lower (0.01-0.02): Subtle enhancement
  • Medium (0.03-0.05): Standard enhancement
  • Higher (0.08-0.15): Strong enhancement for low-contrast images

CLAHE Kernel Size

Range: 4 - 32 (default: 8)

Size of the contextual region for adaptive histogram equalization.

  • Smaller (4-6): More local contrast, may enhance noise
  • Medium (8-12): Balanced local/global contrast
  • Larger (16-32): More global effect, smoother result

Blur Sigma

Range: 0.0 - 5.0 (default: 1.0)

Standard deviation for Gaussian blur (noise reduction).

  • 0.0: No blur
  • 0.5-1.0: Light smoothing (recommended)
  • 1.5-2.5: Moderate smoothing for noisy images
  • 3.0+: Heavy smoothing, may merge nearby features

Neighbor Graph

Graph Type

Algorithm for determining which tubercles are neighbors.

Type Description Edge Count
delaunay Delaunay triangulation - all triangulation edges Most edges
gabriel Gabriel graph - removes edges with points inside diameter circle Moderate
rng Relative Neighborhood Graph - only natural neighbors Fewest edges (recommended)

Recommendation: Use rng for spacing measurements as it best matches the Gayet & Meunier methodology.

Detection Profiles

Profiles are preset parameter combinations optimized for specific image types or species.

Profile Description
default General-purpose parameters for most images
paralepidosteus Large tubercles (5-15 micrometers), wide spacing
lepisosteus Medium tubercles (3-8 micrometers), close spacing
polypterus Small tubercles (1.5-4 micrometers), wide spacing
high-contrast High-quality images with clear boundaries
low-contrast Noisy images with unclear boundaries
scanned-pdf Images extracted from scanned PDF documents

Calibration

Calibration converts pixel measurements to micrometers. There are three ways to set calibration:

  1. Scale bar measurement: Enter the scale bar length in micrometers and pixels
  2. Interactive tool: Draw a line on the scale bar in the image
  3. Direct input: Enter micrometers per pixel directly

Note: Accurate calibration is essential for meaningful measurements. Always use a known scale bar when available.
