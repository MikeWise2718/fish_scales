Keyboard Shortcuts

Quick reference for all keyboard shortcuts in the Fish Scale Measurement UI.

General

Shortcut Action
F1 or ? Open this keyboard shortcuts help
Ctrl + S Save all annotation sets

Set Management

Shortcut Action
Ctrl + N Create a new annotation set
Ctrl + 1-9 Switch to set 1 through 9

Editing

Shortcut Action
Ctrl + Z Undo last edit operation
Ctrl + Y Redo last undone operation
Delete Delete selected tubercle or connection
Escape Cancel current operation / Deselect
Tab Cycle through tubercle selection

Movement & Sizing

Shortcut Action
↑ ↓ ← → Nudge selected tubercle by 1 pixel

Tip: Use the radius slider or +/- buttons in the Edit tab to resize tubercles.

Zoom & Navigation

Shortcut Action
+ or = Zoom in
- Zoom out
0 Zoom to fit image in view

Tip: Scroll wheel zoom can be enabled/disabled in Settings. When disabled, use click-and-drag to pan the image.

Mouse Actions

Action Description
Click tubercle Select tubercle (highlights in table)
Click connection Select connection (highlights in table)
Click table row Select and pan to item on image
Scroll wheel Zoom in/out (if enabled in Settings)
Click + drag Pan image (when zoomed)

Edit Mode Actions

When an edit mode is active (via buttons in the Edit tab):

Mode Mouse Action
Add Tubercle Click on image to place new tubercle
Add Connection Click first tubercle, then click second to connect
Move Click and drag selected tubercle to new position
Delete Multiple Click items to delete them (no confirmation)

Press Escape to exit any edit mode.
